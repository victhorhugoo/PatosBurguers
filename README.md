# :computer: Página web Patos Burguer - frontend
---

## Explicando o projeto

Este projeto teve o objetivo de criar uma página no formato linktree, reunindo todos os perfis da hamburgueria, INstagram, Facebook, Ifood, Cardápio digital e um aba para pesquisa de satisfação que é dividida entre delivery e salão, as respostas da pesquisa geram um brinde para o cliente que resonder pela primeira vez

- React
